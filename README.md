# chose-your-pokemon
pokemon chooser
 ## description 
 This site uses an pokeapi to give you a random pokemon anytime you click on the button!
 ## features 
 - gradient on text
 - animations from tailwind 
 - uses the pokeapi to give you any of the possible pokemon
 -hover effects on almost everything
 ## how to use/set up
 - You can just use the demo link, no further set up is needed. 
 ## tech stack 
 - Next.js
 - Tailwind CSS
 - Typescript
 -PokeApi
 ## why u made it 
 I made it because I have been curious about APIs and I have also been playing pokemon x recently so this was a perfect way to combine them!
 ## learning outcomes (the best part that reviewers love in one's readme, or at least i do)
- I learned how to use APIs also learned a little bit more about deploying on Vercel. I also was able to get more familiar with typescript and tailwind and I am excited to build more advanced projects. This was an entry way project for me to understand how to use APIs and use more typescript/tailwind!
## challenges you faced and how others can avoid repeating the same mistake
I faced an issue with the APIs espeically since I didn't know that the pokemon images were .sprites or that they came in a folder so I spent some time on that, so my recommendation for this is to be sure at what the syntax requires (and looking at documentaion helps a lot). Along with this, I had some issues with deploying and I was able to look at documentation but also other peoples guide for deploying!
